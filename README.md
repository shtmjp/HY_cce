# My Python project template

## How to set up

```
pyenv local 3.11.2 \
&& pipenv --python 3.11.2 \
&& pipenv shell \
&& pipenv install
```

flake8等のdev-packagesがないときは

```
pipenv install --dev
```

flake8等のdev-packagesがないときは

```
pipenv install --dev
```
